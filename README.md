# Fizz-Buzz-NodeJs
## FizzBuzz Problem ##

<p>
Print the numbers incrementing from the initial to N. Unless:
<br>
  * Let be a multiple of 3, then print "fizz"
<br>
  * Multiply 5, then print "buzz"
<br>
  * Be a multiple of 3 and 5 at the same time, then print "fizzbuzz"
</p>
